# CHANGELOG

## [0.0.1] - 2024-06-27

### Added

- Initial setup of the fastapi web-app project, including basic project structure and configuration files. See [README.md](README.md) for more details.
- API endpoints for basic functionality as described in [app-api/src/routers/api.py](app-api/src/routers/api.py) and [app-api/src/routers/hello/hello.py](app-api/src/routers/hello/hello.py).
- Configuration for development environment including `.vscode` settings for Python development and debugging setup in [`.vscode/launch.json`](.vscode/launch.json).
- Recommended extensions for Visual Studio Code in [`.vscode/extensions.json`](.vscode/extensions.json) to enhance development experience.
- Python version specification in [`.python-version`](.python-version) for consistent development environments.

### Notes

- This is the first official release of the fastapi web-app. The project is set up with a focus on development experience, security, and a clear contribution process.
